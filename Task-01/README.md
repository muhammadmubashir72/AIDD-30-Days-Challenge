## 4) Key Concepts to Understand

### AI Turning Point (Oct 2025)
AI became so strong that every developer started using it. Coding, debugging, and testing all involve AI now.

### Agentic AI
Agentic AI can **think, plan step-by-step, take action, and use tools**. It is not just a chatbot, it is a true **AI agent**.

### Evaluation-Driven Development (EvDD)
In EvDD:  
- AI generates code  
- Humans **evaluate the output**  
- Check if it is correct  
- Decide what needs improvement  
- Decide what to test  

This is a new style where **AI writes, humans review**.

### Test-Driven Development (TDD)
TDD is still important in AI era:  
- Write tests first  
- Then write code  
- Ensure tests pass  

Tests make sure AI-generated code works correctly.

### AI Productivity Boom
